# π (Project Intelligence)

Comprehensive EPC Project Management System for "Karimun LNG Terminal - Phase 2" (USD 2.4B).

## Tech Stack
- **Frontend:** Next.js 15 (App Router), React 19, TypeScript
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **State:** Zustand
- **Styling:** Tailwind CSS 4 + Glassmorphism Dark Theme
- **Icons:** Lucide React

## Modules (14)
1. Overview Dashboard
2. HSSE Manager (7 sub-tabs)
3. Planning & Scheduling (4 sub-tabs)
4. Daily Reports
5. Cost Control (EVM)
6. Procurement
7. Quality Control
8. Document Control
9. Engineering
10. Commissioning
11. HR & Admin
12. Finance
13. Security
14. Admin Panel (96-Role Registry)

## RBAC
96 roles across Groups A-O covering all EPC disciplines.

## Getting Started
```bash
npm install
npm run dev
```

## Deploy
Connected to Vercel for auto-deployment on push.
